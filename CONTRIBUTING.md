# Contributing

Thanks for your interest in contributing.

## Scope

This repository is used to track issues related to the Micro-Frontend utilities.
The implementation code is maintained in a separate private repository.

## How to contribute

1. Open an issue describing the bug, proposal, or question.
2. Provide clear reproduction steps and expected behavior when reporting bugs.
3. Keep issue titles short and actionable.

## Pull requests

At the moment, code pull requests are not accepted in this repository because source code is not hosted here.
If this changes, this guide will be updated.
